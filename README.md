Developed a spatio-temporal police incident prediction pipeline using Databricks, Apache Spark, TensorFlow, and scikit-learn. The project applies machine learning and deep learning models to Swedish police open data, combined with explainable AI (SHAP) and Power BI visualizations to analyze incident patterns, interpretability, and model performance within a medallion lakehouse architecture in Unity Catalog with weekly scheduled data extractions with Lakeflow Jobs. Implemented end-to-end engineering and workflows for machine learning including data extraction from a public API, feature engineering, model training, evaluation and explainability analysis. Compared Decision Tree, Random Forest, XGBoost, LSTM, and CNN models for spatio-temporal police incident count prediction, while leveraging SHAP to identify the spatial and temporal factors driving model predictions.
